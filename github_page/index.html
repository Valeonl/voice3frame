Общее описание выпускной квалификационной работы

Эта работа посвящена исследованию и разработке системы автоматизации процесса видеомонтажа с использованием голосового управления. Целью является создание интуитивно понятного интерфейса, позволяющего пользователям эффективно управлять процессом монтажа видео с помощью речевых команд.

Проведенное исследование

Подготовка к исследованию

Подготовка включала анализ существующих решений, определение ключевых сценариев использования голосового управления в видеомонтаже, и разработку прототипа системы для тестирования.

Устройства и материалы

Использовался конденсаторный микрофон [Модель], стандартный ПК с ОС Windows, видеоредактор [Название], библиотека распознавания речи [Название].

Эволюция подхода

Первоначально рассматривался подход с использованием простых ключевых слов, однако он оказался недостаточно гибким. Далее была предпринята попытка внедрения грамматик, что улучшило точность, но усложнило добавление новых команд. Финальная версия использует гибридный подход на основе NLU для лучшего баланса гибкости и точности.
